# 🔮 Future Tasks Prediction (Next 20 Steps)

현재 프로젝트 진행 상황(Phase 3 완료)을 바탕으로, 사용자가 저에게 지시할 가능성이 높거나 프로젝트 완성도를 위해 반드시 필요한 **추가 작업 20가지**를 예측하여 제안합니다.

## 🎨 A. Asset & Creative (비주얼 완성)
1.  **[Core] 215종 크리처 실제 이미지 생성**: 현재 플레이스홀더로 대체된 모든 이미지를 AI로 실제 생성 (`fill_creature_content` 파이프라인 활용).
2.  **[UI] 스킬 아이콘 215종 생성**: 크리처별 고유 스킬에 매칭되는 직관적인 아이콘 제작.
3.  **[Sound] 월드별 전용 BGM 적용**: 아스가르드(웅장), 심연(공포), 무릉도원(동양풍) 등 5개 테마 배경음악 적용.
4.  **[Sound] 전투 타격음 및 피격음 다양화**: 속성별(화염, 냉기, 타격) 사운드 이펙트 차별화.
5.  **[Visual] 전투 스킬 이펙트 강화**: 단순 컷신을 넘어선 실제 파티클(Particle) 효과 구현.

## ⚔️ B. Gameplay Depth (게임성 심화)
6.  **[System] 속성 상성(Elemental Advantage) 구현**: 화>풍>지>수 등 상성 관계에 따른 데미지 ±30% 로직 적용.
7.  **[System] 장비(Artifact) 시스템 추가**: 크리처에게 장착 가능한 무기/방어구 아이템 및 강화 시스템.
8.  **[System] 스킬 레벨업 로직**: 동일 크리처 중복 획득 시 스킬 레벨 상승 및 계수 강화.
9.  **[Content] 월드 보스(World Boss) 레이드**: 전 서버 유저가 협력하여 거대 보스를 처치하는 비동기 레이드.
10. **[Content] 무한의 탑(Tower mode)**: 특정 제약 조건(예: ★3 이하만 입장 가능)이 있는 도전 콘텐츠.

## 📊 C. User Experience (편의성 및 몰입)
11. **[UX] 튜토리얼(Onboarding) 가이드**: 신규 유저를 위한 인터랙티브 튜토리얼 (가챠 -> 전투 -> 강화 흐름 유도).
12. **[UI] 메인 로비 Live2D 효과**: 대표 크리처(호감도 Lv4)가 숨쉬거나 터치에 반응하는 애니메이션 효과.
13. **[UI] 데미지 플로팅 텍스트 개선**: 치명타, 회피, 상성 데미지를 색상과 크기로 시각적 구분.
14. **[Feature] 오프라인 방치 보상**: 게임 종료 후 재접속 시 경과 시간에 따른 골드/경험치 정산 팝업.
15. **[Feature] 도감(Collection) 버프 시스템**: 특정 그룹(예: "아스가르드 신족" 5종) 수집 시 영구 스탯 보너스 부여.

## 🛠️ D. Technical & System (안정성)
16. **[Tech] 세이브 데이터 클라우드 연동**: 로컬 스토리지를 넘어선 Firebase/서버 연동 (가상 구현).
17. **[Tech] 에셋 로딩 최적화**: 215종 이미지의 Lazy Loading 및 메모리 관리 로직 고도화.
18. **[Tech] 안티 치트(Anti-Cheat) 검증**: 전투 결과값 변조 방지를 위한 서버 사이드 검증 로직(시뮬레이션).
19. **[Settings] 다국어 지원(i18n)**: 영어/일본어/중국어 등 로컬라이제이션 JSON 데이터 구축.
20. **[Clean-up] 레거시 코드 리팩토링**: 초기 프로토타입 단계의 하드코딩된 데이터(`StageData` 일부 등)를 완전한 데이터 기반으로 전환.
