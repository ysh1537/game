
# 🎨 Master Image Style Guide (v2.0)

**최우선 원칙 (ABSOLUTE RULE - DO NOT BREAK)**:
1. **전원 여체화 (All-Female / Moe Anthropomorphism)**: 모든 신(Zeus, Odin, Hades 등), 몬스터, 크리처는 **반드시** 아름다운 미소녀 또는 성인 여성으로 의인화되어야 합니다. (남성 캐릭터 금지)
2. **스타일**: **"Blue Archive" + "Nikke"** 스타일의 고퀄리티 2D 셀 셰이딩 일러스트.
3. **매력 포인트**: 글래머러스한 비율(Thick Thighs, Wide Hips)과 매력적인 얼굴(Beautiful Face) 필수.

**타겟**: 성인 서브컬처 매니아 (Adult Subculture Fans)
**최종 검토일**: 2025-12-28 (Updated)

## 1. 🚫 절대 금지 (Negative Rules)
다음 요소가 포함될 경우 **즉시 폐기**합니다.
- **스타일**: 서양 코믹스(Marvel/DC풍), 실사(Realistic), 3D 렌더링, 유화/두꺼운 터치.
- **외형**: 30대 이상으로 보이는 노안, 주름, 못생김, 불쾌한 골짜기.
- **아티팩트**: **텍스트(Text)**, **UI 요소(버튼, 상태창)**, 말풍선, 워터마크, 액자 프레임.
- **구성**: 신체 절단(잘린 머리/팔), 과도한 데포르메(SD 제외), 배경에 묻히는 캐릭터.

## 2. ✨ 핵심 스타일 가이드 (Positive Rules)
- **장르**: 2D 셀 셰이딩(Cell Shaded), 일본 모바일 수집형 RPG 일러스트 스타일 (블루 아카이브, 니케, 에픽세븐 풍).
- **연령/외모**:
  - 남/녀 불문 **아이돌급 외모**와 비율.
  - 피부는 깨끗하고 맑게 표현.
  - 눈은 크고 반짝이며 생기 있게.
- **채색**:
  - 맑고 채도 높은 색감 (Vibrant Colors).
  - 깔끔한 선화(Line Art)와 명확한 명암 경계.
- **구도**:
  - 캐릭터가 돋보이도록 배경과 분리감 확보 (인물 강조).
  - 전신(Full Body) 위주이나 디테일이 살아있어야 함.

## 3. 📂 파일 및 폴더 구조 (Folder Structure)
이미지는 다음 규칙에 따라 엄격하게 분류됩니다.
`images/{Category}/{SubCategory}/{Rank}/`

- **Category**: `creatures`, `backgrounds`, `items`, `ui`
- **SubCategory (Creaturesonly)**: `asgard`, `olympus`, `shangrila`, `abyss`, `wild`
- **Naming**: `{id}_{variation}.png`
    - 기본: `dragon_fire.png`
    - 표정: `dragon_fire_joy.png`, `dragon_fire_angry.png`

## 4. 🔍 품질 검수 체크리스트 (Quality Check)
생성된 이미지는 사용 전 반드시 다음을 확인합니다:
1. [ ] 텍스트나 UI가 없는가?
2. [ ] 얼굴이 젊고 매력적인가? (못생기거나 늙어 보이면 즉시 탈락)
3. [ ] 일본 애니메이션 스타일인가? (서양풍 절대 금지)
4. [ ] 사지(손/발)가 정상적인가?
